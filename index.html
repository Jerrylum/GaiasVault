Frequently Asked Questions

What does the resourcepack do?

The resource pack hides end crystals since there are many of them in the cells.

How long did this take to build?

Ah, 4 weeks.

How to use the map?

Check out the readme file in the world download or watch Deaf's Video (Click Here)

Can you provide schematics file of the prison?

There's a schematic located in the ZIP folder with the world. Please read the Quick Reference Hand Book for more information

How to lock a prisoner?

Check out the Quick Reference Hand Book.

How could a guard leave / enter the prison?

Guards can enter via respawn, stasis, or the guard portal and can exit via guard portal or emergency exit. Please check out the Quick Reference Hand Book.

How does the chunk ban machine work?

The chunk ban machine works by having a dropper repeatedly dispense a skulker full of max size books. When the server tries to send the information in the books to the client, it causes some sort of error which causes them to be kicked.

Is chunk ban cheating?

Ban Books can be made in vanilla survival. It can then just be copied to fill up a shulker box, so it does not count as cheating.

Why chunk ban is not working?

Chunk banning does not work in singleplayer, but if the chunk ban machines are not working on a server, make sure that you have escape mode enabled. The outer chunk bans are disabled and inner chunk bans locked when escape mode is off.

I found a way to escape the prison. What should I do?

You can post the method in the #prison-discussion channel in Deaf's Discord, or you could wait for an event to try it either live on stream or just with trained guards. Make sure that your escape attempt stays within the rules however, as if it does not it does not count.

Escape rules:

  1. If testing in single player, assume that you cannot walk past the chunk bans (if you get a lag spike at some point that's usually a good indicator you walked into one)
  2. Make sure it works in survival, there's no point in trying a creative only escape attempt.
  3. Try not to rely on to much luck. While you can technically chorus all the way to the guard area from the portal room (if you ignore chunkbans) the odds of you actually doing it are low.
  4. Every item you use must be obtainable in 1.16 survival (No god sets that were obtained in 1.12 or something)
  5. If your escape involves using items somewhere past the first bed gate, make sure you actually have a way to get items past the first bed gate.
  6. Make sure you're on at least easy difficulty, but normal or hard is better.
  7. Do not switch out of survival once you start an attack, a possible exception to this rule is if you misplace a block causing you to be trapped, you can then break that block using creative.
